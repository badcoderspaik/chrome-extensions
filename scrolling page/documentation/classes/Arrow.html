jumpDown

Function private

Defined in scripts\Arrow.js:190

Обработчик нажатия кнопки "вниз".
Сбрасывает интервал прокрутки (если таковой запущен) вверх,
устанавливает флаг down_jumping в true для предотвращения повторного запуска интервала прокрутки вниз,
запускает интервал, в котором при каждом вызове функции страница на указанное число пикселей прокручивается вниз.
Если позиция полосы прокрутки window.pageYOffset >= page_height(высота страницы)
(что говорит о том, что страница прокручена в конец),
происходит сброс интервала прокрутки и установка флага down_jumping в false. Флаг должен обязательно быть установлен
в false, в противном случае произойдет блокировка последующих нажатий на кнопку вниз.

jumpUp

Function private

Defined in scripts\Arrow.js:165

Обработчик нажатия кнопки "вверх".
Сбрасывает интервал прокрутки (если таковой запущен) вниз,
устанавливает флаг up_jumping в true для предотвращения повторного запуска интервала прокрутки вверх,
запускает интервал, в котором при каждом вызове функции страница на указанное число пикселей прокручивается вверх.
Если позиция полосы прокрутки window.pageYOffset <= 0 (что говорит о том, что страница прокручена в начало),
происходит сброс интервала прокрутки и установка флага up_jumping в false. Флаг должен обязательно быть установлен
в false, в противном случае произойдет блокировка последующих нажатий на кнопку вверх.

onmousedown

Defined in scripts\Arrow.js:229

Срабатывает при нажатии на кнопки-стрелки "вверх/вниз".
Если класс элемента, на котором произошло событие клика равен 'arrow_up', значит нажата кнопка "вверх".
Если же класс равен 'arrow_down', значит нажата кнопка "вниз".
При нажати кнопки "вверх" флаг 'down_jumping' обязательно должен быть установлен в false, потому как он
снимает и устаналивает блокировку запуска интервавла прокрутки вниз. Так как разблокировка повторного
запуска интервала прокрутки страницы вниз происходит лишь при достижении бегунка полосы прокрутки конца страницы
внутри функции интервала, может случиться, что во время работы интервала прокрутки вниз будет нажата кнопка "вверх"
и бегунок не достигнет конца страницы, что сделает невозможным разблокировку. Таким образом нажатие кнопки "вверх" -
это второе место в коде, где необходимо сбросить флаг 'down_jumping'.
Аналогично при нажати кнопки "вниз" сбрасывается флаг 'up_jumping'.
